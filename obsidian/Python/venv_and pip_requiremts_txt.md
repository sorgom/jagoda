## venv

```shell
$> python3 -m venv <venv dir>
$> source <venv dir>/bin/activate
$> . <venv dir>/bin/activate
```

Sample
```shell
$> python3 -m venv wumpel
$> . wumpel/bin/activate
(wumpel)> 
```

## pip requirements.txt

```bash
$> pip3 freeze > requirements.txt
$> pip install -r requirements.txt
```
